Panduan Integrasi HTTP REST API

Gunakan dokumentasi REST API di bawah ini untuk menghubungkan aplikasi/website Anda (seperti Sistem OTP, Kasir, Registrasi Laravel/PHP, CRM, dll.) ke gateway pengirim WhatsApp ini secara instan dan terjadwal.

POST http://localhost:3001/api/v1/send-message

Endpoint ini memproses pengiriman pesan teks WhatsApp dengan prioritas antrean otomatis (Anti-Ban).

Contoh Request Integrasi Multi-Bahasa:

BASH / cURL Request
curl -X POST http://localhost:3001/api/v1/send-message \ -H "Authorization: Bearer supersecretapikey" \ -H "Content-Type: application/json" \ -d '{ "phone": "62895366583095", "message": "Halo! Kode OTP keamanan Anda adalah: 8842", "session": "default", "priority": "high" }'
PHP (Laravel HTTP Client)
use Illuminate\Support\Facades\Http; $response = Http::withHeaders([ 'Authorization' => 'Bearer supersecretapikey', 'Content-Type' => 'application/json' ])->post('http://localhost:3001/api/v1/send-message', [ 'phone' => '62895366583095', 'message' => 'Halo! Kode OTP keamanan Anda adalah: 8842', 'session' => 'default', 'priority' => 'high' // 'high' mempercepat antrean (Instant OTP) ]); if ($response->successful()) { $data = $response->json(); // Berhasil diproses }
Node.js (Fetch API)
const apiKey = 'supersecretapikey'; fetch('http://localhost:3001/api/v1/send-message', { method: 'POST', headers: { 'Authorization': `Bearer ${apiKey}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ phone: '62895366583095', message: 'Halo! Kode OTP keamanan Anda adalah: 8842', session: 'default', priority: 'high' }) }) .then(res => res.json()) .then(data => console.log('Response:', data)) .catch(err => console.error('Error:', err));
Python (requests)
import requests url = "http://localhost:3001/api/v1/send-message" headers = { "Authorization": "Bearer supersecretapikey", "Content-Type": "application/json" } payload = { "phone": "62895366583095", "message": "Halo! Kode OTP keamanan Anda adalah: 8842", "session": "default", "priority": "high" } response = requests.post(url, json=payload, headers=headers) print(response.json())

Penjelasan Payload Parameter JSON:

Parameter Tipe Wajib Keterangan
phone String Ya Nomor HP penerima (misal: 62895xxx atau 0895xxx). Otomatis dikonversi ke format internasional.
message String Ya Pesan teks yang ingin dikirimkan. Mendukung baris baru (\n).
session String Tidak Nama koneksi pengirim yang terdaftar. Jika diabaikan, menggunakan sesi utama `default`.
priority String Tidak Isi dengan "high" jika pesan mendesak (misal: OTP). Pesan prioritas tinggi akan **didorong ke antrean terdepan** dan dikirim dalam **1-2 detik** (bypassing normal broadcast cooldown). Default: "normal".
